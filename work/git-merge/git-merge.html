Identité Graphique

GitHub souhaitait un système visuel qui pourrait être utilisé année après année, mais aussi évoluer tout en restant original. Nous avons créé une identité graphique flexible ancrée dans la typographie, qui reste cohérente avec le temps, tandis que les images peuvent être réinventées pour de futures conférences.

Red — #FC5146
Blue — #289FFC
Purple — #693CC4
Yellow — #FFDC01
Green — #28EFAA
Orange — #FDA12F

Puisqu’il s’agissait de la première année avec ce nouveau système d’identité visuelle, nous souhaitions construire les visuels autour de quelque chose que tous les utilisateurs de Git reconnaîtraient : le graphique de Git. Le graphique n’est pas seulement un élément familier du produit, il témoigne aussi des principes de la conférence : rassembler des gens et partager des idées.

La page d’accueil utilise ces éléments du graphique de Git afin de guider les visiteurs tout au long de la page, avec une activité linéaire fluide et des variations de couleur inattendues apportant de l’énergie à l’événement. Les séances de la conférence, les conférenciers vedettes et les temps forts ont tous pris la mesure de la vitalité et de l’enthousiasme de l’événement dans leur déroulement.
